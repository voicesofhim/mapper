About the Accelerator Observatory

This fork adapts ContextLab Mapper into a semantic map for accelerator participant research. Each node is an anonymized evidence chunk with precomputed embedding and UMAP coordinates loaded from static JSON.

Ask-the-Map responses are grounded in the bundled synthetic evidence. Syntheses are labeled as inferences and kept separate from source excerpts, participant codes, and supporting themes.

The frontend exposes window.mapperActions so a future LiveKit voice agent can highlight map items, set map lenses, select participants, and open evidence without replacing the Mapper rendering architecture.

Privacy and Interpretation

This demo uses synthetic data only. Production data should preserve consent levels, prefer anonymized excerpts, and avoid diagnostic language about participants.

Source project: This work is forked from ContextLab Mapper and preserves its static domain-bundle loading pattern while changing the content model.
